# Finding the percentage of male and female employees of UFRN

In this project, we identify the gender of each employee and visualize the gender gap using a pie graph. Also we find the first female and male names more comuns of UFRN employees.

## Prerequisites

For this example it is necessary:

* [Jupyter-Notebook](http://jupyter.org/)
* [Pandas](!pip install pandas)
* [rpy2](!pip install rpy2)

## Running the tests

Gender_dictionary.ipynb 
Get the database "names_dictionary.csv" with all  brazilian male and female names released on the website "https://www.dicionariodenomesproprios.com.br/"

UFRN_employees.ipynb 
Extract the name of UFRN employees from Brazilian government portal "Portal da Transparência",  web site  "http://www.portaldatransparencia.gov.br/". Save the database  "UFRN_employees.csv".

Find_Gender.ipynb 
Identify the gender of each employee and visualize the gender gap using a pie graph.
Find the first female and male names more comuns of UFRN employees.
